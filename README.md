# Student-management-system
* Developed a fully functional Student Management System using Python, Tkinter, and SQLite.

* Designed a dual-portal system with separate Admin and Student/User dashboards.
  
* Implemented secure Login, Registration, and Logout functionality for users and admin.

* Created an Admin Panel to manage student records, attendance, and academic results.

* Built features for adding, updating, deleting, and searching student information.

* Developed a Student Portal where users can view attendance, marks, and profile details.

* Used SQLite database for efficient data storage and retrieval operations.

* Integrated role-based access control to restrict admin and student functionalities.

* Designed a modern and user-friendly GUI interface using Tkinter with buttons, frames, and hover effects.

* Added attendance management system for tracking student presence records.

* Implemented result management module for storing and displaying student marks and performance.

* Used Python functions and modular programming for better code organization and maintainability.

* Applied CRUD operations (Create, Read, Update, Delete) for database management.

* Included input validation and error handling to improve system reliability.

* Optimized the project for smooth performance and easy navigation.

** Technologies Used
1.Programming Language: Python

2.GUI Framework: Tkinter

3.Database: SQLite

4.IDE: PyCharm / VS Code
